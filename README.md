# jambandbingo

## Install

### Backend
```
# Setup
pip install -r requirements.txt
cd ./server
python manage.py migrate

# Run the dev server
python manage.py runserver
```

### Client
```
# Setup
cd client
yarn install

# Run the dev server
yarn start
````
